Purpose of this system:

This system will predict the main software engineering role as well as the secondary software engineering role of a student studying at Western Mindanao State University-College of Computing Studies. Furthermore, this system will also allow instructors to generate a group of students who are taking the software engineering course.

Process:

  1. To take the personality type test, click the "click here to take the test" button .It will then redirect the user in an external website to take the test.
  2. Cick the "take the test" in the external website which can be found in the upper right corner of the navigation bar.
  3. After taking the test, the result can be viewed by the user, and after viewing the result, the user should select his/her personality type in the field which consists of 4 letters(e.g. ENTJ) in the field.
