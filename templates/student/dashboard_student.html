Purpose of this system:

This system will predict the main software engineering role as well as the secondary software engineering role of a student studying at Western Mindanao State University-College of Computing Studies. Furthermore, this system will also allow instructors to generate a group of students who are taking the software engineering course.

Process:

  • To start, click on the "Click Me" button above.
  • The system will then request the student to input their grades in respective major courses.It will then output the GPA of the student.
  • The system will now ask the students to take the MBTI personality type test.
  • The system will now ask the students to take the multiple intelligence test.
  • The system will fetch the necessary data which includes the GPA, the personality type of the student, and the multiple intelligence test result. After that, the system will process the inputted data and outputs the predicted main and secondary role.

Software Engineering Roles:

  • Project Manager
  • Lead Programmer
  • UI/UX Designer
  • Business Analyst
