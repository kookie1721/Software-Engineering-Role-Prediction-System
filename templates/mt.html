Purpose of this system:

This system will predict the main software engineering role as well as the secondary software engineering role of a student studying at Western Mindanao State University-College of Computing Studies. Furthermore, this system will also allow instructors to generate a group of students who are taking the software engineering course.

Process:

  1. If you know your multiple-intelligences kindly select that applies to you, otherwise, take the test.
  2. To take the multiple intelligence test, click the "click here to take the test" button .It will then redirect the user in an external website to take the test.
  3. The user can now take the test by answering the questions through scrolling the bar into the given choice.
  4. After taking the test, the result can be viewed by the user, and after viewing the result, the user should input his/her strongest multiple intelligence/s in the field
    Note: Only input the strongest multiple intelligences result.
